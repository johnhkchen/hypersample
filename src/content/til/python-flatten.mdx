<figure>
```python
def flatten(arg):
  if isinstance(arg, list):
    return [x for item in arg for x in flatten(item)]
  return [arg]
```
</figure>

Unpacked, it can look like this:

<figure>
```python
def flatten(arg):
  if isinstance(arg, list):
    flatlist = []
    for item in arg:
      if isinstance(item, list):
        for x in flatten(item):
          flatlist.append(x)
      else:
        flatlist.append(item)
    return flatlist
  return arg
```
</figure>
